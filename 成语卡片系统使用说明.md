# 成语卡片系统 - 使用说明

## 📁 文件结构

```
app/src/main/java/com/example/townapp/data/
├── model/
│   └── IdiomModels.kt          # 数据模型定义
└── repository/
    ├── IdiomRepository.kt      # 成语数据仓库
    └── IdiomTriggerManager.kt  # 触发逻辑管理器

app/src/main/java/com/example/townapp/ui/components/
└── IdiomDataCard.kt            # 成语卡片UI组件
```

## 🎯 核心数据模型

### 1. AcademicReference（学术引用）
| 字段 | 类型 | 说明 |
|------|------|------|
| theoryName | String | 理论名称 |
| researcher | String | 提出者 |
| year | Int | 发表年份 |
| summary | String | 一句话摘要（卡片底部显示） |
| fullText | String | 详细解释（折叠显示） |

### 2. TriggerCondition（触发条件）
| 字段 | 类型 | 说明 |
|------|------|------|
| module | String | 触发模块：food/clothing/housing/cognitive |
| conditionType | String | 条件类型：count/idle/price/usage_rate |
| threshold | Double | 阈值 |
| operator | String | 比较运算符：>/</>=/<=/== |

### 3. IdiomRarity（稀有度）
| 枚举值 | 说明 |
|--------|------|
| COMMON | 常见（基础认知） |
| UNCOMMON | 少见（进阶认知） |
| RARE | 稀有（深度认知） |

### 4. IdiomData（完整成语数据）
| 字段 | 类型 | 说明 |
|------|------|------|
| idiomId | Int | 唯一标识 |
| name | String | 成语名称 |
| traditionalMeaning | String | 传统解读 |
| awakeningMeaning | String | 觉醒解读 |
| dataTemplate | String | 数据填充模板 |
| actionSuggestion | String | 行动建议 |
| academicRefs | List | 学术引用列表 |
| triggerConditions | List | 触发条件 |
| rarity | IdiomRarity | 稀有度 |

## 🚀 使用方法

### 1. 在UI中展示所有成语卡片

```kotlin
import com.example.townapp.data.repository.IdiomTriggerManager
import com.example.townapp.ui.components.IdiomDataCard

// 获取所有成语
val idioms = IdiomTriggerManager.getAllIdiomsWithEmptyData()

// 遍历展示
idioms.forEach { (idiom, data) ->
    IdiomDataCard(
        idiom = idiom,
        dynamicData = data,
        onExpand = { /* 展开时的回调，可用于添加觉醒值 */ }
    )
}
```

### 2. 根据用户数据触发成语

```kotlin
// 衣物模块触发
val clothingIdioms = IdiomTriggerManager.getClothingTriggeredIdioms(
    totalCount = 35,      // 衣物总数
    idleDays = 100.0,     // 闲置天数
    priceRatio = 6.0      // 价格倍率
)

// 食物模块触发
val foodIdioms = IdiomTriggerManager.getFoodTriggeredIdioms(
    storageDays = 10.0,   // 库存天数
    expiryDays = 2.0,     // 过期天数
    impulseRatio = 0.4,   // 冲动消费比例
    junkFoodRatio = 0.35  // 垃圾食品比例
)

// 认知模块触发
val cognitiveIdioms = IdiomTriggerManager.getCognitiveTriggeredIdioms(
    learningStagnation = 200.0,  // 学习停滞天数
    gamblingBias = 3.0,          // 赌博倾向
    impatienceScore = 0.8,       // 急躁得分
    echoChamber = 0.85,          // 信息茧房指数
    screenTime = 6.0             // 屏幕时间（小时）
)
```

### 3. 获取推荐成语

```kotlin
// 获取3个推荐成语
val recommended = IdiomTriggerManager.getRecommendedIdioms(count = 3)
```

### 4. 根据稀有度筛选

```kotlin
import com.example.townapp.data.model.IdiomRarity

// 获取稀有成语
val rareIdioms = IdiomTriggerManager.getIdiomsByRarity(IdiomRarity.RARE)
```

## 📊 数据模板填充

成语卡片支持动态数据填充，模板使用 `{变量名}` 格式：

```kotlin
// 模板示例
val template = "你有 {item_count} 件 {category}，超过了 {threshold} 件阈值"

// 数据填充
val data = mapOf(
    "item_count" to "35",
    "category" to "衣物",
    "threshold" to "30"
)

// 结果："你有 35 件 衣物，超过了 30 件阈值"
```

## 🥗 新增食物批判成语卡片详解

### 11. 山珍海味（UNCOMMON）
**批判点**：野生特供重金属超标、土鸡蛋营养价值未必更高
**学术理论**：
- 食品污染与重金属富集（FAO/WHO, 2003）
- 风险感知偏差（Paul Slovic, 1987）
- 环境激素与内分泌干扰（Theo Colborn, 1996）
**行动建议**：选择有正规溯源的食材，不要被「野生」「特供」迷惑
**触发条件**：wild_food > 1.0

### 12. 油润鲜香（COMMON）
**批判点**：中国菜油脂严重超标，一份菜吃掉一天配额80%
**学术理论**：
- 中国居民膳食指南与油脂摄入（中国营养学会, 2022）
- 味觉适应与阈值变化（Linda Bartoshuk, 1993）
- 烹饪方式与营养损失（International Food Information Council, 2018）
**行动建议**：尝试「少一半油」做法
**触发条件**：high_oil > 1.0

### 13. 精益求精（UNCOMMON）
**批判点**：过度烹饪导致营养严重流失
**学术理论**：
- 食物加工度与营养密度（Carlos Monteiro, 2010）
- 维生素稳定性与烹饪损失（Catherine Ross, 2006）
- 血糖生成指数与食物形态（David Jenkins, 1981）
**行动建议**：试试「粗一点」的烹饪
**触发条件**：overcooked > 1.0

### 14. 原汁原味（UNCOMMON）
**批判点**：「原汁原味」背后是添加剂的高级包装
**学术理论**：
- 鲜味物质与鲜味协同效应（池田菊苗, 1908）
- 食品添加剂的「天然」包装（Marion Nestle, 2002）
- 味觉可塑性与环境影响（Gary Beauchamp, 1997）
**行动建议**：学会辨别「真鲜」和「假鲜」
**触发条件**：artificial_umami > 1.0

### 15. 趁热打铁（COMMON）
**批判点**：65℃以上热食是WHO认证的2A类致癌物，中国食管癌高发元凶
**学术理论**：
- 热食与食管癌风险（IARC/WHO, 2016）
- 中国食管癌筛查与早诊早治指南（2022）
- 食道黏膜损伤研究（中华消化杂志, 2020）
**行动建议**：记住「3分钟原则」，放凉到65℃以下再吃
**触发条件**：hot_food > 65.0

### 16. 因小失大（COMMON）
**批判点**：省5块钱隔夜菜，花500块医药费，浪费生命时间
**学术理论**：
- 隔夜菜亚硝酸盐研究（食品科学, 2019）
- 食源性疾病监测报告（中国疾控中心, 2021）
- 沉没成本谬误（Hal Arkes & Catherine Blumer, 1985）
**行动建议**：少做一点，吃多少做多少
**触发条件**：leftover > 1.0

### 17. 得不偿失（COMMON）
**批判点**：最低消费/吃回本陷阱，用健康换商家利润
**学术理论**：
- 沉没成本谬误（Arkes & Blumer, 1985）
- 损失厌恶（Kahneman & Tversky, 1979）
- 中国急性胰腺炎流行病学报告（2024）
- 商务部令2025年第3号（禁止最低消费）
**行动建议**：宁愿打包也不要吃撑，遇到变相最低消费打12315举报
**触发条件**：over_spend > 30.0 或 overeating > 2.0

## 👕 新增皮肤国成语卡片详解

### 18. 买椟还珠（UNCOMMON）
**批判点**：奢侈品包包99%是品牌溢价，实用价值不到1%
**学术理论**：
- 符号消费理论（Baudrillard, 1970）
- 奢侈品溢价研究（Journal of Marketing, 2022）
- 闲置物品报告（中国消协, 2024）
**行动建议**：包包不超过5个，只留最常用的
**触发条件**：bag_count > 5.0 或 luxury_ratio > 0.5

### 19. 华而不实（UNCOMMON）
**批判点**：首饰0实用价值，重金属超标风险高
**学术理论**：
- 戴比尔斯钻石骗局（Epstein, 1982）
- 首饰重金属污染研究（中华皮肤科杂志, 2021）
- 炫耀性消费（Veblen, 1899）
**行动建议**：首饰不超过3件，只留有纪念意义的
**触发条件**：jewelry_count > 3.0

### 20. 削足适履（COMMON）
**批判点**：为了好看的鞋子伤害脚和膝盖，得不偿失
**学术理论**：
- 中国足部健康调查报告（中国康复医学会, 2024）
- 高跟鞋与膝关节损伤研究（American Journal of Sports Medicine, 2022）
- 进化错配假说（Williams, 1957）
**行动建议**：鞋子不超过8双，优先舒适运动鞋
**触发条件**：shoes_health_cost > 1.0

### 21. 积少成多（COMMON）
**批判点**：袜子内衣囤积成灾，积少成多浪费钱
**学术理论**：
- 心理账户理论（Thaler, 1985）
- 决策疲劳（Baumeister, 1998）
**行动建议**：袜子不超过10双，内衣不超过5件
**触发条件**：sock_count > 10.0 或 underwear_count > 5.0

### 22. 画蛇添足（UNCOMMON）
**批判点**：配饰90%多余，占用空间无实用价值
**学术理论**：
- 简约主义（Marie Kondo, 2014）
**行动建议**：配饰不超过5件，只留真正有用的
**触发条件**：accessory_count > 5.0

## 🧠 学术理论覆盖

系统涵盖30个深度学术理论，涵盖6大学科领域：

### 1. 进化心理学 🧬
| 理论 | 研究者 | 年份 | 核心洞察 |
|------|--------|------|----------|
| 进化错配假说 | Williams | 1957 | 人类心理机制形成于狩猎采集环境，与现代物质社会存在系统性不匹配 |
| 三重脑理论 | MacLean | 1960 | 大脑有三层：爬虫脑（生存本能）、边缘脑（情绪）、新皮层（理性） |
| 镜像神经元 | Rizzolatti | 1996 | 人类大脑中有专门负责模仿和共情的神经元，对「被观察」极度敏感 |

### 2. 行为经济学 💰
| 理论 | 研究者 | 年份 | 核心洞察 |
|------|--------|------|----------|

## 💡 设计理念

1. **深度分层**：表层（成语+解读）→ 中层（学术依据）→ 深层（行动建议）
2. **按需解锁**：用户自主选择是否展开深度内容
3. **数据驱动**：根据用户实际数据动态填充模板
4. **学术支撑**：每个成语都有严谨的学术理论支持
5. **行动导向**：每个成语都配有具体的行动建议
