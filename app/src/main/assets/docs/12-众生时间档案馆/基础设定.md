## 一、功能定位

**核心一句话**：完整承载项目「见证无数普通人一生」的核心定位，是独一份人文公益展示页面。

### 与小镇宪章的契合度

| 宪章条款 | 档案馆落实 |
|----------|-----------|
| 永不加入焦虑驱动设计 | 无排行榜、无热门推荐、无签到打卡 |
| 永不输出评判性结论 | 每种人生只呈现事实，不分好坏优劣 |
| 永不强迫用户行动 | 玩家想来就来想走就走，无召回机制 |

## 二、核心功能

### 1. 分类检索系统

**分类维度（无优先级）**
- 片区：全镇/东区/西区/南区/北区/中心区
- 年龄段：青年（18-35）/中年（36-55）/老年（56+）
- 职业：全职业/教师/医生/工人/农民/店主/程序员/快递员/...

**平等原则**
- 所有分类权重完全一致
- 无热门推荐、无优先排序
- 底部提示：「自由翻阅，每个人都是完整独立的个体」

### 2. NPC 时间线详情页

**时间线节点类型**
- `YEARLY`：年度节点（每5年一个人生阶段）
- `MEMORY`：回忆碎片节点
- `CAREER_CHANGE`：职业变更节点
- `HEALTH_EVENT`：健康事件节点
- `RETIREMENT`：退休节点
- `DEATH`：离世节点

**年度节点描述示例**
- 20岁：开始独立生活
- 25岁：踏入职场
- 30岁：成家立业
- 35岁：责任渐重
- 40岁：上有老下有小
- 45岁：身体开始提醒你
- 50岁：半生已过
- 55岁：孩子长大了
- 60岁：退休
- 65岁：晚年

### 3. 认知分层注解

**高认知（≥4）**
```
每个人的人生节奏不同，快慢没有优劣。
所见的只是概率的一种呈现，不必比较。
```

**中认知（≥2）**
```
每个人都在自己的时区里前行。
```

**低认知**
无注解

**设计原则**
- 极淡金色低透明度小字
- 不干扰主体验
- 只呈现事实，不输出"应该如何活"

### 4. 专属色调动态切换

浏览任意 NPC 时间线时，自动切换专属光影：
- 教师：温柔米白柔光
- 工人：质朴灰黄
- 医生：冷静浅蓝
- 创业者：高对比亮金
- 晚年老人：低饱和暖棕褪色
- 失意者：暗角、低亮度
- 奋斗者：通透高光

### 5. 静默纯画面浏览模式

一键切换，隐藏所有解读文字，仅保留画面。
适合沉浸式体验 NPC 人生轨迹。
