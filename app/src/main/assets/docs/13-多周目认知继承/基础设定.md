## 一、功能定位

**核心一句话**：认知成长永久保留，财富、人际重置，传递「内心沉淀不会被岁月清空」的温和价值观。

### 与小镇宪章的契合度

| 宪章条款 | 多周目落实 |
|----------|-----------|
| 永不加入焦虑驱动设计 | 无多周目成就、奖励、强制重复打卡 |
| 永不输出评判性结论 | 不评判哪种人生选择更好，只拓展叙事包容度 |
| 永不强迫用户行动 | 纯拓展叙事视角，不新增数值内卷 |

---

## 二、核心机制

### 1. 周目重置规则

| 维度 | 一周目 | 二周目+ |
|------|--------|---------|
| 认知等级 | 从 0 开始积累 | **直接 5（智慧者）** |
| 认知碎片 | 积累 | **永久保留** |
| 财富 | 初始值 | **重置为 0** |
| NPC 好感 | 初始值 | **重置为 0** |
| 年份进度 | 从 20 岁开始 | **重新开始** |
| 对话视角 | 随认知解锁 | **直接全部解锁** |

### 2. 二周目开智机制

```kotlin
fun getAvailableDialogueLevel(): Int {
    // 二周目及以上：直接解锁最高智慧视角
    if (_currentRound.value >= 2) {
        return 5  // WISDOM 级别
    }
    // 一周目：使用当前认知等级
    return _cognitionLevel.value
}
```

**效果**
- 二周目玩家直接获得通透的智慧者视角对话
- 鼓励玩家尝试完全不同的人生选择
- 弱化单一标准答案思维

### 3. 进入下一周目

```kotlin
fun startNextRound(startAge: Int = 20) {
    // 1. 先更新认知状态（保存历史最高等级，递增周目数）
    CognitionState.enterNextRound()
    // 2. 重置游戏引擎（财富、NPC 好感、年份进度清零）
    SimulationEngine.resetWithAge(startAge)
}
```

## 三、数据模型

### CognitionState

```kotlin
object CognitionState {
    // 当前认知等级（一周目用）
    private val _cognitionLevel = MutableStateFlow(0)
    val cognitionLevel: StateFlow<Int> = _cognitionLevel.asStateFlow()
    
    // 历史最高认知等级（跨周目保留）
    private val _maxCognitionLevelEver = MutableStateFlow(0)
    val maxCognitionLevelEver: StateFlow<Int> = _maxCognitionLevelEver.asStateFlow()
    
    // 当前周目数（1=一周目，2=二周目...）
    private val _currentRound = MutableStateFlow(1)
    val currentRound: StateFlow<Int> = _currentRound.asStateFlow()
    
    // 认知碎片总数（跨周目累计）
    private val _totalFragments = MutableStateFlow(0)
    val totalFragments: StateFlow<Int> = _totalFragments.asStateFlow()
    
    // 获取当前可用的对话视角等级
    fun getAvailableDialogueLevel(): Int
    
    // 进入下一周目
    fun enterNextRound()
}
```

## 六、设计原则

### 不做的设计

| 禁止项 | 原因 |
|--------|------|
| 多周目成就系统 | 违反宪章第一条 |
| 周目奖励机制 | 违反宪章第一条 |
| 强制重复打卡 | 违反宪章第一条 |
| 数值内卷 | 违反宪章第二条 |
| 召回机制 | 违反宪章第三条 |

### 做的设计

| 设计项 | 目的 |
|--------|------|
| 认知碎片永久保留 | 传递「内心沉淀不会被岁月清空」 |
| 二周目开智 | 鼓励尝试不同人生选择 |
| 财富/好感重置 | 弱化数值积累焦虑 |
| 智慧者对话视角 | 拓展叙事包容度 |

---

## 七、与 NPC 对话系统集成

所有 NPC 对话自动使用 `getAvailableDialogueLevel()`：
- 一周目：随认知等级解锁 surface → observe → wise
- 二周目：直接返回 wise（智慧者视角）

**效果**
- 二周目玩家能看到所有 NPC 的通透对话
- 不需要重复积累认知碎片
- 直接体验「智慧者眼中的众生」

## 随机叙事文案池

- 你睁开眼，又是熟悉的房间。阳光从窗户照进来，和上次一模一样。但你知道——这一次不一样了。
- 你站在人生的十字路口，这场景你好像见过。上次你选了左边，走了很多弯路。这一次，你笑了笑，选了右边。
- 那个人又出现在你面前，说着和上次同样的话。你没有像上次那样激动，只是平静地听着——你知道结局，但你选择温柔地对待。
- 你又走进了这家便利店，店员还是那张脸。上次你在这里买了一包烟，抽了一整夜。这一次，你买了一瓶热牛奶。
- 你看着窗外下雨，和记忆里的某个日子一模一样。上次你在雨里奔跑，浑身湿透，觉得全世界都在和你作对。这一次，你撑了伞。
- 你打开手机，又是那个熟悉的日期。你知道接下来会发生什么——那些你上次后悔的选择，这次你可以做得不一样。
- 你又看到了那个在路边哭泣的年轻人。上次你匆匆走过，觉得那是别人的事。这一次，你停下来，递了一张纸巾。
- 同样的街道，同样的风。你走在路上，脚步比上次轻了很多。因为你知道，所有你曾经害怕的事情，最后都会过去。
- 你看着镜子里的自己，和上次开始时一样年轻。但你的眼睛里，多了一些东西——那是活过一次的人才有的平静。
- 又一次站在起点，你心里没有了上次的慌。你知道哪些坑要绕开，哪些人值得珍惜，哪些事根本不值得生气。
- 你端起饭碗，吃着和上次同样的家常菜。上次你觉得平淡无味，总想着要去远方。这一次，你慢慢嚼着——尝出了米香。
- 熟悉的音乐响起来，你愣了一下。这首歌，在上辈子某个深夜陪过你。你没有像上次那样流泪，只是轻轻跟着哼。
- 又有人来给你"建议"了，告诉你应该怎么活。上次你认真听了，然后走了很久错路。这一次，你笑着说了声谢谢，然后按自己的节奏走。
- 你看着眼前这个要和你争吵的人，心里没有了愤怒。你记得上次你们吵得很凶，两败俱伤。这一次，你选择退一步——不是输了，是懂了。
- 同样的夜晚，同样的失眠。上次你翻来覆去，焦虑未来。这一次，你起来倒了杯水，坐在窗边看月亮——你知道，天总会亮的。
- 你又拿到了第一份工资，和记忆里一样的数字。上次你迫不及待买了最新的手机。这一次，你先给家里打了钱，然后存了一部分。
- 那个机会又来了。上次你因为害怕失败而错过，后悔了很久。这一次，你深吸一口气，伸手抓住了它——哪怕还是会失败。
- 你又站在这个分别的路口，和上次一样的场景。上次你哭着挽留，丢了尊严也没能留住。这一次，你笑着挥了挥手。
- 熟悉的困难又出现在面前，像一座山。上次你觉得天塌了，绕了很远的路。这一次，你卷起袖子——你知道你能翻过去。
- 你看着日历，又是这个日子。上次你在这天做了一个重大决定，后来想想其实没必要那么急。这一次，你多等了三天。
- 同样的批评传到耳朵里，和上次一模一样的话。上次你难受了好几天，觉得自己一无是处。这一次，你听了听——有道理的就改，没道理的就当耳旁风。
- 你又躺在这张床上，准备开始新的一天。上一次，你带着期待和不安。这一次，你带着温柔和笃定——不管发生什么，你都能接住自己。
- 那个曾经让你心动的人又出现了。上次你飞蛾扑火，最后满身是伤。这一次，你只是笑了笑，然后继续走自己的路。
- 同样的饭局，同样的人在吹嘘。上次你自卑，觉得自己不如别人。这一次，你安静地吃菜——你知道每个人都有自己的时区。
- 你又遇到了那个挫折，和记忆里一样疼。但这次你没有崩溃——因为你知道，疼会过去，你会比上次更快站起来。
- 熟悉的风吹在脸上，这条路你好像走过。是的，你走过——但这次你走得更稳，因为你带了上次的记忆和智慧。
- 你看着手里的这张牌，和上次一样。上次你打错了，输得一塌糊涂。这一次，你出牌的手很稳——你知道怎么打了。
- 又到了这个季节，树叶和上次一样落下来。上次你觉得伤感，感叹时间流逝。这一次，你踩着落叶往前走——每一步都算数。
- 你睁开眼，又回到了这里。不是一切归零——是你带着上一次的礼物，重新出发。这一次，你会更温柔地对自己，也对这个世界。
