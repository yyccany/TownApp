## 零、产品定位（一句话）

**VERA 不是成语词典、不是鸡汤文案库，而是一套现代辩证认知工具。**

核心目标：帮助使用者拆解古今俗语、传统道德、网络流行鸡汤里的思维陷阱与时代错位，训练独立判断力；既看清社会结构性规则，也尊重人的情感与自由选择。

设计底色：
- **底层根基**：马克思辩证唯物主义（唯物史观 + 对立统一辩证法）
- **价值内核**：人本思想——人是目的，而非工具
- **发展方向**：面向未来数字化、高生产力文明，用现代科学与现代社会视角重新审视传统观念

## 一、理论根基与五层分析框架

### 1.1 底层主干：马克思唯物史观（第一层）

**核心命题**：物质生产方式（生产力 + 生产关系）决定社会上层建筑（道德、俗语、婚恋、人情、价值观、文化观念）。观念不是凭空诞生，是特定时代生产环境的产物。

**配套核心观点（全程落地）**：

1. **人的本质是一切社会关系的总和**：亲情、友情、爱情、人情全部放在社会生产关系中分析，不孤立谈人性。
2. **辩证看待历史事物**：不全盘否定传统俗语（承认古代生存价值），也不盲从传统（点明现代适配缺陷），坚持一分为二的辩证法。
3. **批判资本异化**：第二层专门拆解消费主义、职场内卷、婚恋市场、人情功利化，是马克思《资本论》中「资本把人的情感、劳动、关系工具化」的现代延伸。

**关键区分**：不是教条化背诵理论，是落地为普通人能看懂的词条解读。不搞空洞政治论述，只用来拆解日常俗语、生活鸡汤。

### 1.2 现代社会结构批判（第二层）

**理论来源**：西方马克思主义 / 法兰克福学派（马尔库塞、弗洛姆）  
**核心命题**：工业文明、消费社会会制造「单向度的人」，用世俗道德、鸡汤、消费话术束缚个体自由。

**专门拆解**：
- 职场用「年轻多吃苦」无偿压榨劳动力
- 婚恋市场用「门当户对、结婚才算完整」物化亲密关系
- 社交用「多个朋友多条路」把友情功利化

### 1.3 现代认知心理学（第三层）

**理论来源**：20 世纪至今主流实证心理学  
**核心命题**：人类思维存在系统性偏差，这些偏差是客观实验证实的漏洞，解释人为什么容易被洗脑、盲从老话、陷入自我内耗。

**8 类核心认知偏差标签**（词条必须挂载至少一个）：
1. 非黑即白思维
2. 幸存者偏差
3. 归因谬误
4. 损失厌恶
5. 社会对比偏差
6. 确认偏误
7. 框架效应
8. 从众效应

### 1.4 自然科学实证（第四层）

**核心命题**：世界运行有客观物质规律，区分「文学比喻」和「客观现实」。

**专门破除**：
- 玄学、宿命论（无客观验证依据）
- 伪养生、偏方食补（营养学、微生物学、生理学检验）
- 长生不老幻想（海夫利克极限、端粒衰老机制、古代丹药重金属毒性）
- 点石成金幻想（原子核物理、元素嬗变客观成本）

**写作要求**：不空谈科学精神，务必给出可验证的具体科学事实。

### 1.5 主体性情感与人际价值（第五层）

**理论来源**：马克思人本思想 + 现代存在主义  
**核心命题**：人是目的，不是工具。补齐理性解构的短板，完整看待亲情、爱情、友情、人情善恶。

**三个判断标尺**：
1. **自愿性**：是否出于双方的真诚自愿
2. **边界感**：是否尊重彼此的独立边界
3. **付出平衡性**：是否存在单向剥削或道德绑架

**正向**：双向自愿的亲密联结拥有独立精神价值，不能完全简化为激素、利益、生存工具。  
**负向**：客观剖析情感异化后的控制、PUA、道德绑架、功利社交、人性自私与恶意。  
**最终做到**：既能理性看透情感骗局，也依然拥有真诚爱人的能力。

### 1.6 重要理论边界：辩证唯物主义 vs 机械唯物主义

| 维度 | 机械唯物主义（避免） | 辩证唯物主义（遵循） |
|------|---------------------|---------------------|
| 人的本质 | 只看生理物质，忽略社会关系 | 社会关系总和 + 物质基础 |
| 情感 | 爱情 = 多巴胺，亲情 = 基因自私 | 既承认物质底层，也承认社会关系与自主选择的精神价值 |
| 传统 | 全盘否定或全盘接受 | 一分为二，具体时代具体分析 |
| 未来观 | 宿命论或乌托邦 | 承认客观规律，肯定人的主观能动性 |

## 二、词条字段结构模板（7 个模块，全部必填）

每条词条严格按以下 7 个模块组织。不可缺省，不可调序。

### 字段一览

| 序号 | 字段名 | 用途 | 字数建议 |
|------|--------|------|----------|
| 1 | 标题 + 收藏星标 | 词条入口标识 | 标题 ≤ 10 字 |
| 2 | 分类标签 + 认知偏差胶囊标签 | 快速定位类别和思维漏洞 | 标签 2-5 个 |
| 3 | 基础定义 (traditionalMeaning) | 时代溯源，解释观念诞生的生产力与社会背景 | 100-200 字 |
| 4 | 现实案例 (examples) | 两组对照：古代合理场景 vs 现代滥用规训场景 | 每组 80-150 字 |
| 5 | 小镇辩证解读 (townPerspective) | 五层框架核心解读 | 300-500 字 |
| 6 | 可操作校准技巧 (actionableTip) | 无命令式说教，至少两种中立可选方案 | 150-250 字 |
| 7 | 常见误读 + 核心启示 (distortedTruth + keyMessage) | 逐条对应认知偏差，一句话双向辩证收尾 | 200-300 字 |

### 2.1 标题 + 收藏星标

- 标题为词条所涉俗语 / 成语 / 流行语本身
- 右侧收藏星标图标，用户自行操作
- 不在内容规范层面干涉

### 2.2 分类标签 + 认知偏差胶囊标签

**分类标签**（必选其一）：
- 认知误区
- 焦虑陷阱
- 片面认知
- 底层认知
- 传统观念

**认知偏差胶囊标签**（至少挂载一个，从 1.3 节 8 类中选取）：  
直接标注对应的思维漏洞类型。

### 2.3 基础定义

**写作红线**：
- 只做时代溯源，不添加主观评价
- 解释该观念诞生的生产力条件、社会结构、当时合理性
- 禁止使用「愚昧」「落后」「洗脑」等情绪化贬损词
- 语气：中性、客观、陈述事实

**模板句**：  
「在 XX 时代背景下（具体生产力条件），由于 XX 社会结构特征（具体生产关系），产生了 XX 观念。这一观念在当时的历史条件下，承担着 XX 社会功能。」

### 2.4 现实案例（严格两组对照）

**古代合理场景**：
- 还原观念诞生时期的典型应用场景
- 说明在当时条件下为何可行、为何合理

**现代滥用规训场景**：
- 揭示该观念在今天如何被滥用为控制工具
- 必须基于社会结构变化（非主观感受）说明为何不再适用

**写作红线**：
- 两组对照必须同时出现，不可偏废
- 古代场景不可过度美化（那是生存刚需，并非理想社会）
- 现代场景不可无限放大（必须针对具体可验证的社会现象）

### 2.5 小镇辩证解读（五层框架核心段落）

这是词条的灵魂部分。按以下结构逐层展开：

```
【历史溯源】（第一层：唯物史观）
→ 该观念诞生于何种生产力与生产关系条件

【现代异化】（第二层：社会结构批判）
→ 该观念在今天如何被资本/职场/婚恋市场/舆论工具化

【思维陷阱】（第三层：认知心理学）
→ 人们容易因哪些思维偏差盲目接受该观念

【科学检验】（第四层：自然科学）
→ 如有伪科学成分，用可验证事实破除；如无科学争议，可省略此层

【情感辩证】（第五层：主体性情感人际）
→ 该观念涉及的情感关系，何为正向价值，何为异化控制
```

**写作要求**：
- 五层必须完整（第四层无争议时可省略，但须注明「无科学争议，不适用第四层」）
- 每层 60-100 字，总计 300-500 字
- 禁止空洞的政治论述，必须落到具体可感知的生活场景
- 辩证收尾：既看到问题，也给出出路

### 2.6 可操作校准技巧

**核心原则**：无命令式说教。不告诉用户「你应该怎么做」，只提供中立可选方案。

**格式要求**：
- 至少提供两种行动参考方向
- 每种方案以「可以尝试」「可以选择」「也可以」开头
- 选择权完全交给用户
- 禁止使用「你必须」「你应该」「最正确的方式是」

**示例结构**：
```
方案一：可以尝试 XX（适合 XX 情况）。
方案二：也可以选择 XX（如果 XX 条件允许）。
提醒：以上仅为参考路径，没有唯一正确解。
```

### 2.7 常见误读 + 核心启示

**常见误读**（distortedTruth）：
- 逐条对应该词条的认知偏差标签
- 每条一句话说清误读表现
- 至少 2 条，至多 5 条

**核心启示**（keyMessage）：
- 一句话双向辩证收尾
- 格式：「既 XX（避免极端 A），也 XX（避免极端 B）」
- 必须同时包含批判和建设两个方向
- 字数 ≤ 50 字

## 三、写作红线与禁用话术清单

### 3.1 绝对禁止

| 类别 | 禁止内容 | 原因 |
|------|----------|------|
| 政治标签 | 「资产阶级」「社会主义优越性」「阶级斗争」「反动」等 | 产品面向普通人，不做政治站队 |
| 人身攻击 | 「愚蠢」「无知」「脑残」「被洗脑的」等对观念持有者的贬损 | 批判观念，不批判人 |
| 年龄歧视 | 「老年人顽固」「年轻人幼稚」等 | 认知偏差不受年龄限制 |
| 性别偏见 | 「女人就该」「男人必须」等性别角色固化 | 违背平等观 |
| 绝对真理 | 「只有 XX 才是对的」「永远不要 XX」 | 违反辩证原则，也违反选择权交给用户的原则 |
| 情感否定 | 「爱不过是激素」「亲情只是基因策略」 | 机械唯物主义，违背第五层 |
| 命令式 | 「你必须」「你应该」「正确做法是」 | 违反工具属性，规训式表达 |
| 玄学兜底 | 「信则有不信则无」「命里有时终须有」 | 违背第四层科学实证 |

### 3.2 需谨慎使用

| 类别 | 允许但有条件 |
|------|------------|
| 「内卷」 | 必须说明具体机制，不可作为万能标签 |
| 「PUA」 | 必须给出明确行为模式描述，不可泛指一切不满 |
| 「消费主义」 | 必须落实到具体消费场景和资本逻辑，不可空洞批判 |
| 「资本」 | 严格限定在经济关系分析，不可泛化为一切负面事物的代名词 |
| 「规训」 | 必须说明规训主体、机制、后果 |

### 3.3 情感类词条额外红线

涉及亲情、爱情、友情的词条，额外遵守：
1. 不彻底解构为纯物质/纯利益关系
2. 必须包含「自主选择权」视角
3. 不替用户做关系决策（如「应该分手」「应该原谅」）
4. 承认情感伤害的真实性，但不用受害者叙事强化无力感
5. 辩证收尾：既保护自己，也不彻底封闭

## 四、查重校验标准

### 4.1 同类词条区分规则

当两个词条题材相近时，必须满足以下至少一项区分：
1. **时代错位不同**：应用的古代场景与现代社会场景不同
2. **认知偏差不同**：对应的思维漏洞类型不同
3. **情感五层侧重不同**：虽然都涉及亲情，但一个侧重宗族集体，一个侧重个体独立
4. **科学检验不同**：一个涉及伪科学成分，另一个不涉及

### 4.2 重复判定标准

出现以下任一情况，判定为重复，必须合并或删除其一：
- 标题同义（含近义成语）
- 解读的认知偏差完全相同，且现实案例高度重合
- 核心启示 (keyMessage) 完全相同

### 4.3 新词条入库前检查清单

- [ ] 标题不与已有词条同义
- [ ] 7 个模块完整无缺
- [ ] 分类标签已标记
- [ ] 认知偏差标签已挂载（至少 1 个）
- [ ] 五层解读完整（第四层可注明略过）
- [ ] 校准技巧至少 2 种方案
- [ ] 无禁用话术
- [ ] 古代 / 现代案例两组对照齐全
- [ ] 核心启示双向辩证收尾
- [ ] 无命令式表达
- [ ] 无与其他词条的实质性重复

## 五、版本迭代与修改规范

### 5.1 框架调整

- 新增理论层次（如本次 v3.0 新增第五层情感视角）→ 同步更新本手册、同步回填已有词条
- 修改分类标签体系 → 同步更新所有词条的标签
- 调整字段结构 → 同步更新所有词条的字段格式

### 5.2 词条迭代触发条件

出现以下情况时，已有词条必须复审：
1. 框架理论层次变更（如新增第五层）
2. 认知偏差标签体系更新
3. 社会现实发生重大变化（使原案例过时）
4. 接收用户有效反馈（非单纯立场不同，而是事实错误或逻辑漏洞）

### 5.3 修改纪录

每次修改本手册，必须在文档末尾追加：
